# Media Saber Skills

Media Saber 官方技能包集合。

## 🎯 支持的技能

- **Media Saber MCP** - 完整MCP工具集（28个工具）用于订阅、站点操作、推荐、云任务和系统检查
- **订阅助手** - 订阅工作流优化
- **站点操作** - 站点签到、数据同步、诊断
- **推荐助手** - 电影和电视推荐系统
- **云助手** - 云盘传输和离线下载
- **极空间监控** - NAS系统状态和硬件健康检查

## 🚀 快速开始

### 在 OpenClaw 中导入

使用以下配置文件之一：

- [OpenClaw 标准配置](media-saber-mcp/examples/openclaw.mcpServers.json)
- [Claw 兼容配置](media-saber-mcp/examples/claw-compatible.mcpServers.json)

复制配置到 OpenClaw 工作区的 `mcpServers` 设置中。

### 通过 ClawHub 安装

访问 [clawhub.ai](https://clawhub.ai) 搜索安装你需要的技能。
